# 📊 Zudio Multi-Store Sales Intelligence Dashboard

## 🚀 Project Overview
This project demonstrates an end-to-end data analysis workflow using Microsoft Excel. The dashboard transforms raw retail data into meaningful insights related to sales performance, regional trends, and category contribution.

Using Power Query for data cleaning and Pivot Tables for analysis, an interactive dashboard was created to support data-driven decision-making.

💡 This project highlights my ability to work with real-world data, perform structured analysis, and present insights effectively.

---

## 🛠 Tools Used
- Microsoft Excel
- Power Query
- Pivot Tables
- Data Visualization

---

## 📈 Key Features
- KPI Cards (Sales, Orders, Profit Margin)
- Monthly Sales Trend Analysis
- Category-wise Distribution
- State & City Performance
- Interactive Dashboard using Slicers

---

## 🔍 Key Insights
- Sales are concentrated in specific cities and states
- Balanced demand across categories
- Seasonal variations in sales trends
- High Average Order Value indicates premium transactions

---

## 📊 Business Recommendations
- Expand in low-performing regions
- Focus on high-performing markets
- Optimize inventory based on demand
- Use targeted marketing strategies
